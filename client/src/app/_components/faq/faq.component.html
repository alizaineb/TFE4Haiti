Inscription/Connexion

Rendez-vous sur la page se connecter.
La section "Demande de compte" permet d'envoyer une demande d'accès à l'application.
Pour la demande vous devez obligatoirement fournir:
  • Une adresse e-mail valide
  • Un prénom
  • Un nom
  • Un rôle qui peut être:
    • Administrateur
    • Chercheur
    • Employé

Une fois la demande envoyée, un administrateur devra valider votre inscription.

Une fois le demande validée, un lien vous sera envoyé sur votre e-mail pour que vous puissiez choisir un mot de passe.
Si votre demande est refusée, un e-mail vous sera envoyé avec une explication si l'administrateur en a précisée une.
Rendez-vous sur la page se connecter.
Dans la section "Se connecter", entrez votre e-mail et le mot de passe que vous choisi.
Rendez-vous sur la page récupérer mon compte. Un lien vous sera envoyé pour mettre à jour votre mot de passe.

Station

Rendez-vous à la page Carte ou Mes stations et recherchez votre station. Pour afficher son détail, appuyer sur le bouton:

Rendez-vous à la page Mes stations et cliquez sur le bouton

Pour ajouter une station il faut obligatoirement:
  • Un nom
  • Une intervalle
  • Une commune
  • Un bassin versant
  • Une latitude
  • Une longitude
  • Une date de création

Pour la latitude/longitude, vous pouvez directement cliquer sur la map afin de placer votre station.

Vous pouvez ajouter optionnellment:
  • Une altitude
  • Une première note

Une fois que vous avez validé votre station, un administrateur doit valider celle-ci.

  • Rendez-vous à la page Mes stations et cliquez sur le bouton: de la station à modifier.
  • Rendez-vous sur le détail de la station et cliquez sur le bouton

Vous n'avez pas le droit de modifier l'intervalle d'une station.
Cette fonctionnalité est réservé à l'adminstrateur.
